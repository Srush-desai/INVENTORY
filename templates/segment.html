📊 About the Dataset

The dataset used is Mall_Customers.csv consisting of 200 entries from a mall. It includes:

  • CustomerID – Unique identifier
  • Gender, Age
  • Annual Income (k$)
  • Spending Score (1–100) – Assigned by the mall based on customer behavior

🔍 What We Did

We applied K-Means Clustering (an unsupervised learning algorithm) using scikit-learn to group customers based on:

  • 📈 Annual Income
  • 💳 Spending Score

These clusters help businesses identify and target specific customer segments like budget buyers, premium spenders, and average shoppers.

🛠 Technologies Used

  • 🧠 Machine Learning: K-Means (Scikit-learn)
  • 🐍 Backend: Python + Flask
  • 📊 Data Handling: Pandas, CSV files
  • 🎨 Frontend: HTML + CSS (Dark Mode)
